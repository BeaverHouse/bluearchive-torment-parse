# BA Torment (Raw Data)

총력전/대결전 데이터는 [해당 링크][ref0]에서 원본 데이터를 받아 Python으로 가공하고 있습니다.  
⚠ 문제시 링크가 삭제될 수 있습니다.

캐릭터 정보는 [Schale.gg][ref1] 소스에서 받아오고 있습니다.

<br/>

## make venv & Package Install

```
python -m venv .venv

# For Windows Powershell
& .venv/Scripts/Activate.ps1

pip install -r requirements.txt
```
